## Project Overview
This project analyzes COVID-19 time-series data for India. 
The objective was to study case trends, daily growth, and moving averages.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Analysis Performed
- Data cleaning and filtering
- Daily case calculation
- 7-day moving average
- Trend visualization
- Statistical summary

## Key Insights
- Confirmed cases show steady exponential growth.
- Daily case spikes highlight pandemic waves.
- 7-day average smooths fluctuations for better trend analysis.
- Minor negative daily values reflect real-world data corrections.

## Conclusion
The analysis demonstrates how time-series techniques can be used to understand pandemic growth patterns and reporting behavior.
